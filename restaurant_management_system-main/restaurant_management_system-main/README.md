# Restaurant Management System

This project is a Restaurant Management System designed to streamline various operations within a restaurant, including employee management, supplier management, category management, product management, sales tracking, billing, and more.

## Features

- Employee Management: Allows for adding, updating, deleting, and searching employee details such as name, email, contact, gender, date of birth, date of joining, password, user type, address, and salary.
- Supplier Management: Manages suppliers, including details such as name, email, contact, address, and more.
- Category Management: Manages product categories.
- Product Management: Manages dishes or products offered by the restaurant.
- Bill Management: Contains all the bills.
- Billing: Generates bills for customers.
- User Authentication: Provides login functionality with different user roles (e.g., admin, employee).

## Technologies Used

- Python: Backend development and scripting.
- Tkinter: GUI library for building the user interface.
- SQLite: Database management system for storing data.
- Pillow: Python Imaging Library for image processing (e.g., displaying logo images).
- tkcalendar: Calendar widget for date input.
